Montfauçon
A custom font by Caveras

Montfauçon is a serif pixel font recreation based on the original
main font appearing in the video game Broken Sword: The Shadow of
the Templars (and its sequel Broken Sword II: The Smoking Mirror),
developed by Revolution Software and released for PC by Virgin
Interactive in 1996.

The font file in this archive was created by Caveras using FontStruct -
the free, online font-building tool. This font has a homepage where
this archive and other versions may be found:
https://fontstruct.com/fontstructors/caveras
